# Insertion Sort  
Write a function that takes in an array of integers and returns a sorted version of that array. Use the Insertion Sort algorithm to sort the array.

## Example
__Sample Input__:  
```
array = [8, 5, 2, 9, 5, 6, 3]
```
__Sample Output__:  
```
[2, 3, 5, 5, 6, 8, 9]
```
